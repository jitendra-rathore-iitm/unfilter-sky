# Unfilter Sky

## Description
Unfilter sky is a app where we can weather temparature, it is using openweathermap api for data. It show temparature, Humidity and wind speed (Km/h).

Use `git clone https://github.com/jitendra-rathore-iitm/unfiltered-sky.git` to install all the files which are present here. after that use live server extention in vs code to see in web browser or i already deployed it with the help of vercel you can see there also.
